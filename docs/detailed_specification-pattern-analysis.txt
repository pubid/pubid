## SPECIAL HANDLING REQUIREMENTS

### 1. N Number Formatting
- Zero-padded to 3 digits: N001, N002, N004, N009, N041, N044
- Variable length: 1-3 digits (but displayed as 3)
- Must preserve leading zeros in output

### 2. Part Number Handling
- Format: N{number}-{part}
- Example: N0009-1, N0009-2
- Part numbers are simple integers (1, 2, etc.)

### 3. C Range Notation
- Different pattern from N notation
- Format: C{start}-{end}
- Represents a range of numbers
- Examples: C155-168, C199-276

### 4. Base Numbers
Standard BSI base numbers:
- 9074, 9111, 9114, 9210, 9300, 9305, 9611
- All 4-digit numbers
