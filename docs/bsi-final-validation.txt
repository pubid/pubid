6. KNOWN LIMITATIONS
=============================================================

1. Multi-level parts (e.g., "-2-1"):
   - Currently parsed as combined value "-2-1"
   - Enhancement pending for proper separation into part and subpart
   - Not a blocker for current functionality

2. Detailed specification prefix:
   - "N 50/1" format not fully supported
   - Parser expects "BS" prefix for consistency
   - Represents edge case in classification system

3. Some fixture failures (177/1579):
   - May include edge cases, obsolete formats, or data quality issues
   - 88.79% coverage is strong for production use
   - Failures should be reviewed for potential parser enhancements
